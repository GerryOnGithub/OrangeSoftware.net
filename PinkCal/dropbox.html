Sharing pinkcal appointments across two computers

Setup:

  1. Get and install Dropbox (you'll love it)
  2. Configure PinkCal (on both computers) to backup and restore to the Dropbox folder

Usage:

Run the PinkCal backup to save data to the Dropbox folder and later run the restore command on the other computer.

Note that "restore" just adds/imports data from the backup file, it doesn't replace or remove anything.

License

If you have a valid PinkCal license then you are okay to use PinkCal on two computers for your personal use.
