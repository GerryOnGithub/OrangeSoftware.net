Backup

If you want to understand what PinkCal does for backups read this whole page.

Easy Backup

  1. Use the PinkCal backup command to save your day planner entries to floppy disk or to another folder.
  2. Open the To-Do list and print it, or you can use copy-to-clipboard and paste to Notepad.
  3. After installing the new version use the PinkCal restore command to restore your day planner entries.
      

Manual Backup (before version 7.0)

  1. Use Windows Explorer to browse to the folder where PinkCal is installed.
  2. Look for the file PinkCal.data and copy the file to your backup media (floppy/USB drive/CD, etc.)
    If you don't see PinkCal.data then look for backup.month where month is the current month.
      
    Note: backup.month is just a copy of PinkCal.data for the given month.

Manual Backup (starting with version 7.0)

  1. Use Windows Explorer to browse to the folder C:\Documents and Settings\[your name]\Application Data\Pink Calendar
  2. Copy all the files from that folder to your backup media (floppy/USB drive/CD, etc.)
