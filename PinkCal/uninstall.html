Un-Install


To uninstall use the Windows Add/Remove Programs utility.

For older versions of Pink Calendar on Win95/98/Me follow these  instructions:

  1. Click here to download a small un-install program.
     When you are asked where to save the install program, a good place is something like c:\Temp
     
  2. Run the un-install program you just downloaded.
     
  3. After the un-install is complete use File Explorer to delete the Pink Calendar folder: c:\Program Files\Pink Calendar. Delete the un-install program.

Optional manual un-install:

  1. stop the program
  2. delete the Pink Calendar program folder
  3. select Start - Settings - Taskbar and go into Start Menu/Advanced and remove shortcuts to Pink Calendar
